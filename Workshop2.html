Introduction

This workshop of R for Marine Science builds on functional programming and data wrangling skills, focusing on extracting meaningful biological patterns from complex ecological datasets.

2.2 Tidying data

How we make our dataset tidy is by following three interrelated rules:

- Each variable must have its own column.

- Each observation must have its own row.

- Each value must have its own cell.
