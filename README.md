# MB5370_Module_02
### R for Marine Science
This module explores concepts of R for Marine Science through four workshops:

1. Foundations of data science: Wrangling and plotting
2. Advanced data wrangling: Extracting ecological signals from noisy systems
3. Advanced data communication and open-ended sourcing
4. Spatial engineering and reproducible reporting

Ultimately, the goals of this module are to:
- Implement advanced data engineering pipelines to clean, restructure, and validate highly volatile, real-world marine datasets.
- Master the art of publication-grade data visualization by deconstructing and replicating complex figures layer-by-layer.
- Understand coordinate reference systems (CRS) and handle vector-based spatial data frames natively within open-source programming frameworks.
- Transition from standalone script writing to compiling fully automated, reproducible dynamic reports suitable for professional scientific consulting.
